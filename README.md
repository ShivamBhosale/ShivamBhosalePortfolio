# Shivam Bhosale | Data Analyst & AI Engineer Portfolio

A futuristic, interactive portfolio website designed to showcase my work in Data Analysis and AI Engineering. The site features a cyberpunk aesthetic with immersive audio and visual effects.

## 🚀 Features

*   **Immersive Design**: Cyberpunk-inspired UI with neon accents and glassmorphism effects.
*   **Interactive Background**: Dynamic neural network particle animation using HTML5 Canvas.
*   **Audio Engine**: Custom sound effects for hover and click interactions (with toggle).
*   **Visual Effects**:
    *   Glitch text effects
    *   Typewriter animations
    *   Custom cursor with blending modes
    *   Scroll reveal animations
*   **Responsive**: Fully optimized for desktop and mobile devices.

## 🛠️ Tech Stack

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom properties (variables), Flexbox/Grid, Keyframe animations.
*   **JavaScript**: Vanilla JS for the audio engine, canvas animations, and interactivity.

## 📂 Project Structure

```
├── index.html    # Main structure and content
├── style.css     # Styling, animations, and responsive design
├── script.js     # Logic for audio, canvas, and interactions
└── README.md     # Project documentation
```

## 🚀 Getting Started

1.  Clone the repository or download the files.
2.  Open `index.html` in any modern web browser.
3.  Turn up your volume slightly for the full experience!
